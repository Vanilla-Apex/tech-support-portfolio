# Technical Support & Infrastructure Operations Portfolio

Practical IT support, troubleshooting, and infrastructure-focused case studies by Tom Burris.

This repository documents structured technical investigations across Windows, Linux, networking, hardware, SaaS access, and simulated data center scenarios. Each case study follows a support-oriented format: symptoms, initial checks, diagnostic steps, root cause, resolution, prevention notes, and escalation considerations.

The portfolio is designed to show practical troubleshooting discipline, clear technical documentation, and operational thinking applicable to IT support, infrastructure support, field support, and data center operations.

> Note: Data center scenarios are lab-researched simulations designed to demonstrate infrastructure troubleshooting logic, documentation structure, and operational readiness thinking.
>
> ## Skills Demonstrated

- Structured troubleshooting and root cause analysis
- Windows support and system recovery
- Basic Linux service, disk, and log investigation
- DNS, DHCP, TCP/IP, VPN, and local network troubleshooting
- Hardware diagnostics, component replacement, and device testing
- Technical documentation and support-ticket style reporting
- Escalation notes, prevention steps, and operational handover writing
- Infrastructure operations concepts including rack validation, airflow, power redundancy, and link checks
